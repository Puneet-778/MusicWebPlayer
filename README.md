# MusicWebPlayer
# To scrape songs info -
 Song title, song cover image, artist, and song url
 
 1) run npm init -y
 2) npm i puppeteer
 3) npm i fs
 4) node getSongs.js


# Functionalities in webplayer
  1) Play/Pause, volume controls
  2) Switch to any track 
  3) Create playlist
  4) Edit playlist
  5) Delete Playlist
  6) Local storage is used to retain playlist data
